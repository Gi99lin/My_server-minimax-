# Настройка VPN через 3x-ui панель

## Шаг 1: Откройте SSH туннель

```bash
ssh -L 2053:localhost:2053 root@37.60.251.4
```

Пароль: `Tanquzsacjev5hywvo`

## Шаг 2: Откройте 3x-ui панель

Откройте в браузере: `http://localhost:2053/panel`

**Логин:** (проверьте в контейнере или используйте стандартный `admin`)
**Пароль:** (проверьте в контейнере или используйте стандартный `admin`)

## Шаг 3: Создайте новый Inbound

1. Перейдите в **Inbounds**
2. Нажмите **+ Add Inbound**
3. Заполните поля:

### Основные настройки:
- **Remark:** `iPhone VPN`
- **Protocol:** `vless`
- **Listen IP:** оставьте пустым
- **Port:** `443` (или `10443` если 443 занят)

### Client Settings:
- **ID (UUID):** нажмите кнопку генерации или используйте существующий
- **Email:** `iphone@vpn`
- **Encryption:** `none`
- **Flow:** оставьте пустым

### Transport Settings:
- **Network:** `tcp`
- **Header Type:** `none`

### Security:
- **Security:** `none` (или `reality` для лучшей маскировки)

## Шаг 4: Сохраните и экспортируйте

1. Нажмите **Save**
2. В списке inbounds найдите созданный
3. Нажмите на иконку **QR-code** или **Export**
4. Отсканируйте QR-код в FoXray на iPhone

## Шаг 5: Подключитесь

1. Откройте FoXray
2. Импортируйте профиль (QR-код или ссылка)
3. Подключитесь
4. Проверьте интернет

## Если 3x-ui панель не открывается:

Проверьте, что контейнер запущен:
```bash
ssh root@37.60.251.4
docker ps | grep 3x-ui
docker logs 3x-ui --tail 20
```

Если нужно перезапустить:
```bash
docker restart 3x-ui
```
